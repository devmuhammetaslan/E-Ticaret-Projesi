# 🛒 Dinamik E-Ticaret Sepet Modülü

Bu proje, modern web geliştirme pratikleri kullanılarak hazırlanan, sıfırdan kodlanmış dinamik bir e-ticaret sepeti uygulamasıdır. Proje şu an **geliştirme aşamasındadır** ve yeni özellikler eklenmeye devam etmektedir.

## 🚀 Özellikler (Mevcut Durum)
- Ürünlerin statik HTML yerine JavaScript ile dinamik olarak ekrana basılması.
- Sepete ürün ekleme, miktar artırma/azaltma ve sepetten silme algoritmaları.
- `Array.find()` ve `Array.filter()` metotları ile hatasız durum (state) yönetimi.
- Eş zamanlı toplam fiyat ve adet hesaplaması.
- Yandan kayarak açılan (Off-canvas) modern sepet arayüzü (DOM Manipülasyonu).

## 🛠️ Kullanılan Teknolojiler
- HTML5
- CSS3 (Flexbox, CSS Animasyonları)
- Vanilla JavaScript (ES6+, DOM Manipülasyonu)
